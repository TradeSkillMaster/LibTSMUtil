BinarySearch

The BinarySearch module provides an implementation of the binary search algorithm for quickly finding an element within a sorted sequence. The BinarySearch.Table API is provided for easily searching through a table, as well as a BinarySearch.Raw API for searching an arbitrary data sequence via a specified function.

Performs a raw binary search.

Parameters:

  • numEntries (number) – The total number of entries

  • searchValue (<V: string | number>) – The value to search for

  • valueFunc (fun(index: number, ...any): <V: string | number>) – A function to call to get a comparable value for the specified index

  • ... (any) – Extra values to pass to valueFunc
